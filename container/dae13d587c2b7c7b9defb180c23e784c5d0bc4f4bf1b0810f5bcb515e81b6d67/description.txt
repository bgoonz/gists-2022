Explain the differences between imperative and declarative programming.  Answer These two types of programming can roughly be summarized as:  Imperative: how to achieve something Declarative: what should be achieved A common example of declarative programming is CSS. The developer specifies CSS properties that describe what something should look like rather than how to achieve it. The "how" is abstracted away by the browser.
